1. What is Java?
Java is an high-level,object oreinted platform independent programming language
It was developed by James Gosling in 1991
It can also known as platform as it provide it's own JRE & API

2. What is Platform?
A plaform is the hardware or software enviorment in which a piece of software is executed

3. What is Java Virtual Machine(JVM)?
JVM is a virtual machine that enables the computer to run the java program
JVM acts like a run-time engine which calls the main method present in java code
Java code is compiled by JVM to a byte code which is machine independent and close to native code

4.What is java Run-time enviorment(JRE)?
It is a implementation of JVM and JRE is a set of software tool which are used for developing java applications

5.What is Java Development Tool Kit(JDK)?
It is a software development environment which is used to develop Java applications and applets

6.What is Just in time Compiler(JIT)?
It is used to improve the performance. JIT compiles parts of the bytecode that have similar functionality at the same time, and hence reduces the amount of time needed for compilation.

7.What gives Java its 'write once and run anywhere' nature??
Java compiler converts the Java programs into the class file (Byte Code) which is the intermediate language between source code and machine code. This bytecode is not platform specific and can be executed on any computer.

8.What is Object Cloning?
It is an ability to recreate an object entirely similar to an existing object is known as object cloning

9.What is Package?
A package is a collection of related classes and interfaces providing access protection and namespace mangement

16.What is inheritance?

17.What is Polymorphism?
Polymorphism is nothing but assigning behavior or value in a subclass to something that was already declared in the main class.Simply, Polymorphism takes more than one form

  • Compile-time Polymorphism: It allows you to create more than one function with same name and different signature
  • Run-time Polymorphism: It allows you to create more than one function with same name and same signature
